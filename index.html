Welcome to Piano Learner.

Piano Learner is a free open source piano learner web application where the user can learn and record various Piano Tunes.

How To Use

1) Either use your mouse to press the piano keys or

2) Use your keyboard (A to Enter) to play the piano

3) In the learn mode, you can try replaying existing tones in the website

4) In case you play wrong notes, you will have to restart over again, it will help you learn the tone
